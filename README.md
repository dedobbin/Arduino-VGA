# Arduino-VGA-ram
Arduino sketch to to display data from RAM chip on VGA screen
<br><br>
![](https://nop.koindozer.org/gh/ram-vga.png)

- 2 470Ω resistors
- 2 68Ω resistors
- 23LCV1024 chip (RAM)

# Dependencies
[VGAX](https://github.com/smaffer/vgax)
